Easy way to display teams using WPBakery Page Builder.
== Description ==
Team Builder is a WPBakery Page Builder extension that would allow user to display teams using WPBakery page builder.It contains 10 different styles with external linking.You can add unlimited any social icons.It is really simple and easy to setup.It's purely CSS3 based and responsive on all devices.

Note: This plugin requires <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=labibahmed" target="_blank">Visual Composer</a>.<br>

<h3>Features</h3>

<ul>
	<li>10 Different Styles</li>	 
	<li>Border Styles</li>	 
	<li>External Linking</li> 
	<li>100% Responsive</li> 
	<li>Unlimited Social Icons</li>

</ul>
